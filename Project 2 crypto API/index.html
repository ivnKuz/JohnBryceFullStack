CRYPTONYTE

Hello, my name is Ivan Kuzovatov. I am aspiring full-stack developer. This is a college project I made, it uses crypto currency API(coingecko), crypto compare API and charts.js CDN. This project let's you choose out of almost 11 thousand crypto coins and compare their value live in US$(up to 5 coins) or just look at one coin's value in real time.


Technologies used in this project:
JavaScript, TypeScript, CSS, HTML, jQuery, JSON, AJAX, Bootstrap, Caching, Google fonts, Coingecko API, Crypto compare API, Charts.js CDN
